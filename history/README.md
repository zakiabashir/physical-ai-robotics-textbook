# Project History

This directory contains the historical documentation and records of the Physical AI Robotics Textbook project.

## Structure

### `/adr/`
Architecture Decision Records (ADRs) - Important architectural decisions and their rationale

### `/authentication/`
Authentication system implementation history
- OAuth setup and configuration
- JWT token management
- Login/logout flows
- Authentication debugging and fixes

### `/build/`
Build system and deployment setup history
- Initial build configuration
- Build tools setup

### `/chat/`
Chat system and RAG (Retrieval-Augmented Generation) implementation
- Chat widget development
- RAG system integration
- Session management
- Token expiration handling

### `/deployment/`
Deployment guides and status reports
- Railway deployment configuration
- Vercel deployment setup
- Environment configuration
- Deployment troubleshooting

### `/fixes/`
Bug fixes and issue resolutions
- Frontend integration fixes
- Mixed content issues
- API endpoint fixes
- General troubleshooting

### `/implementation/`
General implementation notes and guides

### `/prompts/`
Prompt History Records (PHRs) - AI interaction history organized by feature:
- `constitution/` - Project constitution updates
- `general/` - General maintenance tasks
- `[feature-name]/` - Feature-specific development history

## Maintaining This Directory

When adding new documentation:
1. Place it in the appropriate category folder
2. Use descriptive filenames with dates
3. Update this README if adding new categories
4. Keep old records for reference - don't delete unless absolutely necessary
